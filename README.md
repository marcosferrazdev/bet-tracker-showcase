## 🚀 About

BetTracker is a SaaS platform designed to track and analyze sports betting performance, helping users make data-driven decisions.

This project focuses on scalability, performance, and user experience.

---

## ✨ Features

- User authentication (Email & Google)
- Interactive dashboard
- ROI and performance tracking
- Bet history management
- Secure multi-user data (RLS)

---

## 🧠 Highlights

- Clean Architecture
- UI/UX focused
- Scalable structure
- Real-world SaaS approach

---

## ⚔️ Tech Stack

- React
- Next.js
- TypeScript
- Spring Boot
- Supabase
- PostgreSQL
- Tailwind CSS
